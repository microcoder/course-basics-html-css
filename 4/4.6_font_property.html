Свойство font

Доступны следующие свойства в указанном порядке:

font: variant style weight size / line-height family

Обязательными из этих свойств являются size и family, без них ваша запись font работать не будет:

font: variant style weight size / line-height family

Рассмотрим примеры:

body { font: 14px; /* Не будет работать */ font: italic Arial; /* Не будет работать */ font: 14px Arial, sans-serif; font: 14px / 1.2 Arial, sans-serif; font: italic bold 14px Times New Roman; font: small-caps italic 700 14px / 20px Arial, sans-serif; }
