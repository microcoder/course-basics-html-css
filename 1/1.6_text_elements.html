Элемент <em> (emphasis)

Элемент предназначен для акцентирования текста:

Cats are cut animals.

Элемент <strong>

Элемент предназначен для обозначения важной информации:

Внимание! Идут учения!

Элемент <mark> (marked text)

Элемент помечает текст как выделенный:

C++, Python, js - классные языки программирования

Элемент <small>

Элемент предназначен для различной уточняющей информации. Например, для условий и правовых ограничений в рекламе.

Скидка на все товары 50%

При условии покупки на сумму более 1000 рублей

Элемент <q> (quote)

Элемент предназначен для выделения цитат или обозначения прямой речи. Не создает до и после разрывы как элемент blockquote

Le general Koutouzoff, -- сказал Болкон ударяя на последнем слоге zoff, как француз

Le general Koutouzoff,
-- сказал Болкон ударяя на последнем слоге zoff, как француз

Элементы <ruby>, <rt>

Элементы предназначены для добавления небольшой аннотации сверху или снизу от заданного текста:

A[ei] B[bi] C[si]
エディタРедактор

Элементы <samp>, <kbd>

  • sample output - используется для отображения текста, который является результатом вывода компьютерной программы или скрипта.
  • keyboard - используется для обозначения текста, который набирается на клавиатуре или для названия клавиш.

Нажмите CTRL+S на клавиатуре.

Компьютер должен ответить: Document was saved successfuly

Элемент <code>

Элемент предназначен для отображения одной или нескольких строк текста, который представляет собой программный код. Этот элемент не сохраняет все переносы строк, так как это делает элемент <pre>. Лучше всего элемент <code> вкладывать в элемент <pre>.

function Panel(element, canClose){ this.element = element; this.canClose = canClose; } 

function Panel(element, canClose){
    this.element = element;
    this.canClose = canClose;
}

Элементы <sup> (superscript) и <sub> (subscript)

Элементы обозначают верхний/нижний индекс. Например, для обозначения степени:

Формула эквивалентности массы и энергии: E = mc2

Формула воды: H2O

Элемент <br> (break line)

Элемент устанавливает перевод строки в том месте, где он находится:

Варкалось. Хливкие шорьки
Пырялись по неве,
И хрюкотали зелюки,
Как мюмзики в мове.

Элемент <wbr> (word break opportunity)

Элемент указывает браузеру место, где допускается делать перенос строки в длинном слове.

В этой строке содержится длинное слово которое может быть перенесено браузером, в приемлемых, а не произвольных местах: метоксихлордиэтиламинометилбутиламино

Элемент <span>

Универсальный текстовый элемент, который позволяет изменить вид содержимого через стили:

Яндекс.Браузер предостерегает пользователей, когда они начинают вводить пароль на подозрительных страницах.
