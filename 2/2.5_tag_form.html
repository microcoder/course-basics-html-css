Форма обмена данными

Элемент <form>

Элемент предназначен для обмена данными между пользователем и сервером:

Атрибут action

Указывает обработчик к которому обращаются данные формы при их отправке на сервер:

Атрибут method

Сообщает серверу о типе запроса. Значения атрибута:

  • get - Предназначен для получения требуемой информации и передачи данных в адресной строке (ограничение в 4kb). Это значение метода по умолчанию. Преимущества: Можно сохранить строку запроса в закладках.
  • post - Посылает на сервер данные в запросе браузера. Большие объемы данных используются в форумах, почтовых службах, заполнении базы данных, при пересылке файлов и т.д. Преимущества: Отправка данных свыше 4kb, отправка секретных данных которые не должны быть сохранены или отображаемы в адресной строке браузера.
