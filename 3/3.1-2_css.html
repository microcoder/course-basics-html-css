Структура CSS

Разметка

HTML - это язык разметки.
CSS - это язык стилей, определяющий отображение HTML (и других) документов. CSS работает о шрифтами, цветом, полями, строками, высотой, шириной, фоновыми изображениями, позиционированием элементов и многими другими вещами.

Основные этапы развития CSS

1. CSS уровень 1 (1996-1999) - параметры шрифтов, цвета, ...
2. CSS уровень 2 (12.05.1998) - блочная верстка, селекторы, ...
...
Развитие остановилось на многие годы в силу доминирования браузера Internet Explorer, который не обновлялся годами
...
3. CSS уровень 2.1 (07.06.2011)
4. CSS уровень 3 находится в стадии разработки - трансформация, анимация, ...
5. CSS уровень 4 разрабатывается с 29.09.2011

Анатомия CSS

селектор {
правило
}

Селекторов может быть несколько, тогда они разделяются запятой.
Правило - это свойство со значением. Правил может быть несколько, тогда они разделяются через точку с запятой.
Комментарий возможен через конструкцию /* */

селектор1, селектор2, селектор3 { /* Это комментарий */ свойство: значение; свойство: значение; свойство: значение; свойство: значение; }

Пример кода:

body { background: #f1f1f1; } h1 { font-size: 34px; color: red; text-align: center; } h2 { font-size: 24px; } h1,h2 { font-family: Tahoma; } p { font-size: 18px; color: gray; font-family: Arial; }
