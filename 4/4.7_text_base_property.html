Базовые свойства для работы с текстом

Что такое текст?

Текст (от лат. textus «ткань; сплетение, связь, сочетание») — зафиксированная на каком-либо материальном носителе человеческая мысль; в общем плане связная и полная последовательность символов.

Выравнивание

Осуществляется с помощью свойства text-align и имеет следующие значения:

.uno { text-align: left; } .dos { text-align: start; } .tres { text-align: right; } .quatro { text-align: end; } .cinco { text-align: center; } .seis { text-align: justify; }

"Украшение" текста

Устанавливается с помощью свойства text-decoration, которое имеет три законных значения: overline, line-through и underline. Оно позволяет нам сделать так:

  • Подчеркивание над текстом
  • Зачеркнутый текст
  • Подчеркнутый текст. Можно использовать для ссылок

text-decoration не наследуется дочерними элементами, и им нельзя отменить "украшение", которое применено к родителю.

Текст надчеркнут. Текст попрежнему надчеркнут.

Трансформация текста

Свойство text-transform позволяет менять регистр букв с помощью трех значений: capitalize, lowercase и uppercase.

Допустимо использовать любые единицы измерения длины и их отрицательные значения.

Цвет

Цвет текста задается с помощью свойства color.
Принято использовать hex для сплошных цветов и rgba() для полупрозрачных.

Тень текста

Тень текста задается с помощью свойства text-shadow с такой записью:

text-shadow: offset-x offset-y blur-radius color

Обязательными являются только сдвиг по x и по y. Цвет наследуется от родителя, а blur-radius будет равен 0.

text-shadow: offset-x offset-y blur-radius color
