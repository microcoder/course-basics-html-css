Отношения

div div - Если мы запишем div<пробел>div, то мы выберем все div'ы на странице, а затем выберем все div'ы которые являются потомками ранее выбранных div'ов.
В спецификации CSS Selectors Level 4 (CSS4) планируется закрепить явно:
Selector Relation Description
div >> div Потомок Выбираем все div'ы, затем все div'ы которые являются потомками ранее выбранных div'ов
div > div Сын Выбираем все div'ы, затем внутри этих div'ов выбираем только тех, которые являются div'ами
div + div Брат Выбираем все div'ы, затем только первый элемент следующий за данным div'ом
div ~ div Братья
