Директива DOCTYPE

Первой строчкой в документе директива указывает спецификацию по которой создан этот документ

!DOCTYPE html

Тег <html>

Тег задает границу для всего документа. Пройти валидацию HTML кода можно по этому адресу

Тег <title>

Тег определяет заголовок документа. Единственный обязательный элемент, и он не должен быть пустым

CERN

Тег <link>

Тег устанавливает связь с внешними документами

Атрибут rel (relationship)

Атрибут rel (relationship) тега позволяет описывать тип связи между документом и внешним файлом. Значений у этого атрибута довольно много, все они доступны по этому адресу. Значение alternate - указывает, что у документа есть альтернативный формат.

Тег <base>

Тег определяет базовый адрес для всех ссылок документа, а также как они будут открываться (в текущем окне target="_self" или в новом окне target="_blank")

Тег <script>

Тег предназначен для описания скриптов, может содержать ссылку на скрипт или его текст на определенном языке.

Тег <style>

Тег предназначен для определения стилей Атрибут media описывает устройство, для которого следует применять стилевое оформление. Значения атрибута media могут быть: all, screen, print, speech, braille

Тег <meta>

Тег определяет метаинформацию документа. Эту информацию могут использовать поисковые роботы, браузер или даже социальные сети.

OpenGraph

OpenGraph - возможность связывать свой контент с социальными сетями, возможность там его правильно представлять. Этих свойств довольно много
