HTML элемент

HTML элемент состоит из открывающего тега и закрывающего . Внутри тега указываются атрибуты, которые позволяют задавать свойства элемента. Между открывающим и закрывающим тегом находится содержимое этого элемента.

HTML разметка

HTML разметку можно разбить на два типа:

  1. В теге <head> находится описательная часть.
    Определяет мета информацию документа. Этой информацией пользуются браузеры, поисковики, социальные сети

  2. В теге <body> находится структурная часть.
    Это визуальная часть, которую видит пользователь.
