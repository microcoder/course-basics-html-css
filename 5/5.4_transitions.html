Переходы

Простой переход

Переходы - это анимация от одного набора CSS свойств к другому.
Для перехода необходимо:

  • Два набора свойств:
    • начальный набор свойств (color: #f00)
    • конечный набор свойств (color: #00f)
  • Свойство transition - описание свойств и характеристик анимации перехода
  • Инициатор - действие, которое вызывает изменение от одного набора свойств к другому (:hover, :target, :focus, :active)

Ракета

Мы хотим, чтобы у нас взлетала ракета. Но как взлетает ракета? Вначале она набирает скорость, т.е., движится медленно, а потом в какой-то момент времени она ускоряется. Мы хотим, чтобы анимация длилась 3 секунды и указываем функцию cubic-bezier.
Дополнительно можно посмотреть CSS код на примере шарика с гелием.

Краткая запись

Записывать свойства для переходов в отдельности бывает не совсем удобно:

.long { transition-property: transform; transition-duration: 3s; transition-delay: 1s; transition-timing-function: ease-in; }

Поэтому, существует короткая запись, и для этого используется свойство transition:

.short { /* transition: property duration [timing-function] [delay] */ transition: transform 0.5s ease-in 1s; }

Можно указать несколько свойств, они указываются через запятую:

.multi-short { transition: transform 0.5s ease-in, background-color 0.5s ease-in 1s; }

Можно записать еще короче, указав значение all, в этом случае будут анимироваться все возможные свойства. Также, указываем длительность анимации, все остальные свойства остаются по умолчанию:

.super-short { transition: all 5s; }
